## 2. 录屏前准备

环境要求：

- Apple M4 16GB 验证机。
- Ollama `0.20.7`，已安装 `gemma3:4b`。
- ComfyUI `0.25.1`，已安装 Flux.2 klein 4B BF16、Qwen FP4 text encoder 和 Flux2 VAE。
- `config/comfyui-workflow.json` 已从示例复制。

构建应用：

```bash
cd /Users/dc/Desktop/localart-canvas-ops
npm ci
cp -n config/comfyui-workflow.example.json config/comfyui-workflow.json
npm run make
```

确认 Ollama：

```bash
ollama serve
ollama list
```

若 Ollama 已作为后台服务运行，只执行 `ollama list`。

启动本机现有 ComfyUI：

```bash
cd /Volumes/2Tnd/LocalArt/ComfyUI
.venv/bin/python main.py --listen 127.0.0.1 --port 8188
```

## 3. Electron 内 M1 闭环录屏步骤

用下面命令启动刚构建的未签名应用，确保录屏使用固定 Ollama 模型：

```bash
cd /Users/dc/Desktop/localart-canvas-ops
OLLAMA_MODEL=gemma3:4b \
"out/LocalArt Canvas-darwin-arm64/LocalArt Canvas.app/Contents/MacOS/localart-canvas"
```

录屏按以下顺序操作：

1. 展示 loading 窗口随后进入主画布，证明应用从动态 loopback HTTP 服务加载。
2. 展示右侧“服务状态”：Ollama 与 ComfyUI 均为“已连接”，并展示实际 Canvas 路径。
3. 在空白画布上创建一个矩形或点击 `Add AI placeholder`，确认对象处于选中状态。
4. 在 `Revision request` 输入：`Create a minimalist red circle centered on a clean white background, soft studio light, no text.`
5. 点击 `Send to Ollama`，等待面板出现 Agent 回复和 `gemma3:4b` 模型名。
6. 点击 `Generate revision`，等待 Flux.2 klein 完成生成；Apple M4 首次推理约需 60 秒。
7. 展示新 `AIImageHolder` 出现在原对象右侧，并且新图可选中、移动和删除。
8. 展示生成文件已落盘：

```bash
find "$HOME/Library/Application Support/localart-canvas/canvas/assets" \
  -maxdepth 1 -type f -print -exec file {} \;
```

## 4. 录屏后补充截图

### 4.1 Electron 退出不停止用户服务

先正常关闭 Electron，再执行：

```bash
pgrep -fl "localart-canvas|LocalArt Canvas Helper" || echo "LocalArt processes stopped"
curl -fsS http://127.0.0.1:11434/api/tags >/dev/null && echo "Ollama alive"
curl -fsS http://127.0.0.1:8188/system_stats >/dev/null && echo "ComfyUI alive"
```

截图应同时显示：无残留 LocalArt 进程，Ollama 与 ComfyUI 仍可访问。

### 4.2 从 `userData/canvas` 恢复

重新启动同一个 `.app`，截图展示刚生成的图片、位置和选中状态已恢复；右侧 Canvas 路径应为：

```text
/Users/dc/Library/Application Support/localart-canvas/canvas
```

### 4.3 未签名应用的 Gatekeeper 路径

在 Finder 中找到 `LocalArt Canvas.app`，右键选择“打开”，记录应用可启动。若开发机仍阻止打开，可执行：

```bash
xattr -dr com.apple.quarantine "/Applications/LocalArt Canvas.app"
```

该命令只用于本机开发测试；M2 第一阶段不包含签名或公证。
